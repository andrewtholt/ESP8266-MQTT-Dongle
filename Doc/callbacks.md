# Callbacks

## Set callbacks
**WIFI_CONNECTED**

If command from serial, return command error.

Places a message into the serial out queue.
On success 
```
SET WIFI_CONNECTED TRUE
```
On failure
```
SET WIFI_CONNECTED FALSE
```

**MQTT_CONNECTED**
If command from serial, return command error.

Places a message into the serial out queue.
On success 
```
SET MQTT_CONNECTED TRUE
```
On failure
```
SET MQTT_CONNECTED FALSE
```
